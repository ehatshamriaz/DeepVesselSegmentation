# Deep Vessel Segmentation for Fundas images


This repository provides code for evaluating retinal vessel segmentation methods and reproducing the results presented in the associated research paper.

The work is based on the study by Li Ding, Ajay E. Kuriyan, Rajeev S. Ramchandran, Charles C. Wykoff, and Gaurav Sharma titled:

“Weakly-Supervised Vessel Detection in Ultra-Widefield Fundus Photography via Iterative Multi-Modal Registration and Learning”, published in IEEE Transactions on Medical Imaging (2020).

DOI: https://doi.org/10.1109/TMI.2020.3027665

## 📦 Dependencies

The program requires **Python 3.x** and the following packages:

- PyTorch v0.4.1  
- torchvision v0.2.1  
- OpenCV-Python  
- NumPy  
- scikit-learn  
- scikit-image  
- Pillow  
- segmentation-models-pytorch  

For specific package versions and compatibility details, please refer to the **Code Ocean capsule**.

> ⚠️ GPU is highly recommended.
